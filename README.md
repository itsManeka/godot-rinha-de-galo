
# Rinha de Galo

Ganhe dinheiro nas rinhas de galo e derrote os 5 campeões para se tornar o Rei dos Galos.

Aa Setas movem o galo e espaço ataca.

Obs.: Não foi adaptado para dispositivos móveis.

Jogo realizado para a [#JamSecreto](https://itch.io/jam/1jam-secreto/rate/867089) da [@Jamtastic](https://twitter.com/jamtasticgd).
